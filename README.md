# 사용법

1. git clone 을 받고 test.sh파일을 서브젝트 디렉토리로 옮김
2. test.sh파일을 열어서
	1. PROJECT변수에 서브젝트의 실행파일 이름을 넣음
	2. TEST DIRECTORY 변수에 테스트할 케이스가 든 디렉토리 이름을 넣음
3. 서브젝트의 실행파일이 있는지 확인하고 test.sh파일 실행 시킴

기본적인 틀이고, 수정해서 사용하셔도 됩니다.
